# Bloque Simple (bloque_simple)

el mod añade un bloque de broma ,y puede ser para decoracion,el cual al romperse genera un sonido de "el monte Everest no tiene nada en contra de mi" con la foto de mi gato fan de goku,es un mod de prueba que 0uedes usar con tus aligos o en tu servidor:D

## Compatibilidad
- Luanti (Minetest) 5.x o superior
- Funciona en servidores multijugador: Sí

## Dependencias
- **Obligatorias:** ninguna
- **Opcionales:** ninguna

## Instalación
1. Descarga o clona este repositorio.
2. Copia la carpeta en `/minetest/mods/`.
3. Activa el mod desde el menú del juego o en `minetest.conf`:
