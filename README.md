## Name: Simple Block (bloque_simple)

This mod adds a joke block that can also be used for decoration.
When broken, it plays a sound clip of "Mount Everest has nothing on me"
along with a photo of my Goku fan cat. It's a test mod that you can use
in any context, and it will still be weird no matter what.

## Compatibility
- Luanti (Minetest) 5.x or higher
- Multiplayer server support: Yes

## Dependencies
- **Required:** none
- **Optional:** none

## Installation
1. Download or clone this repository.
2. Copy the folder to `/minetest/mods/`.
3. Enable the mod from the game menu or in `minetest.conf`.

## Features
- Adds a new joke block
- bloque_simple
